# Analysis Reports

This folder contains all analysis files generated during the performance benchmarking and GPT-5 evaluation session.

## Contents Overview

### Markdown Reports
- **FINAL_GPT5_EVALUATION_REPORT.md** - Final comprehensive evaluation report using GPT-5
- **RUN_GPT5_EVALUATION.md** - Documentation of the GPT-5 evaluation run
- **COMPREHENSIVE_PER_COMMIT_ANALYSIS.md** - Detailed analysis of each commit
- **DETAILED_PER_COMMIT_ANALYSIS.md** - Additional per-commit analysis details
- **RESEARCH_ANALYSIS.md** - Research findings and insights
- **CRITICAL_ANALYSIS.md** - Critical findings and issues identified

### JSON Data Files
- **gpt5_evaluations.json** - GPT-5 evaluation results
- **all_attempts_comprehensive.json** - All attempts data
- **all_commits_analysis.json** - Analysis of all commits
- **commit_attempt_summary.json** - Summary of commit attempts
- **complete_commit_analysis.json** - Complete commit analysis data
- **comprehensive_per_commit_analysis.json** - Per-commit comprehensive analysis
- **comprehensive_run_analysis.json** - Run-level comprehensive analysis
- **detailed_commits_sample.json** - Sample of detailed commit data
- **detailed_pattern_analysis.json** - Pattern analysis results
- **detailed_run_observations.json** - Detailed observations from runs
- **per_commit_detailed_analysis.json** - Detailed per-commit data
- **per_run_detailed_analysis.json** - Detailed per-run data
- **run_analysis_data.json** - Raw run analysis data
- **run_by_run_analysis.json** - Run-by-run breakdown
- **sample_commit_analysis.json** - Sample commit analysis
- **stdout_analysis.json** - Analysis of stdout logs
- **top_commits_for_analysis.json** - Top commits identified for analysis

### Scripts
- **evaluate_patches_with_gpt5.py** - Script used to evaluate patches using GPT-5

## Purpose

These files were generated to analyze the performance of the OpenHands agent system, evaluate patch quality using GPT-5, and identify patterns in successful and failed commits.

## Usage

For the main evaluation report, start with `FINAL_GPT5_EVALUATION_REPORT.md`.

For detailed technical analysis, refer to the JSON files and per-commit analysis reports.
